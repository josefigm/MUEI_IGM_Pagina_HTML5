La página se ha realizado siguiendo la estructura semántica de HTML5. Los
elementos visuales que forman parte de dicha estructura se pueden ver en 
detalle en la fotografía de la página que se incluye en este directorio,
marcados con un número cuya circunferencia es roja.

ELEMENTOS NOVEDOSOS

	Los elementos novedosos de HTML5 que se han incluido son el elemento 
	<video> y el elemento <canvas>.

	Elemento <video>:

	Como su nombre indica, sirve para mostrar un video de manera nativa en
	una página web. Para que esta experiencia sea gratificante y 
	personalizable para el usuario se añaden algunas propiedades:

	autoplay:	hace que el video se reproduzca automaticamente al 
			cargar la página. Para que funcione en chrome hay que
			añadir la siguiente propiedad.

	muted:		hace que el video se reproduzca sin audio. 
			Imprescindible para lograr el autoplay en chrome.

	controls:	incluye controles tales como un botón de play, control
			de sonido para que el usuario pueda controlar el video.
			
	Luego dentro del elemento video existe un elemento source en donde se 
	le indica la ubicación del video y el tipo del mismo.

	Elemento <canvas>:

	El elemento HTML <canvas> se puede usar para dibujar gráficos mediante 
	código en JavaScript. Por ejemplo, se puede usar para hacer gráficas, 
	composiciones fotográficas, crear animaciones, o incluso procesado o 
	renderizado de vídeo en tiempo real. En este caso, se ha utilizado para
	dibujar una cara "a trazo".

	Para el elemento canvas se ha utilizado la siguiente propiedad:

	id:		Identifica univocamente el elemento en la página.
